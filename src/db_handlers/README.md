# Err code

### 300 - 400: mongo_client.rs

- **300**: MongoDBクライアントオプションの初期化に失敗
- **301**: MongoDBクライアントの作成に失敗
- **302**: データベース名が空
- **303**: データベースロックの取得に失敗
- **304**: データのBSON変換に失敗
- **305**: ドキュメントの挿入に失敗
- **306**: 挿入されたIDがObjectIdではない
- **307**: データベースロックの取得に失敗
- **308**: ドキュメントの検索に失敗
- **309**: ドキュメントが見つからない
- **310**: データベースロックの取得に失敗
- **311**: ドキュメントの検索に失敗
- **312**: ドキュメントが見つからない
